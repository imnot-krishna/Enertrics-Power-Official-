## Image Requirements

- **Product Images**: 800x800px, JPG format
- **Blog Images**: 1200x630px, JPG format
- **Team Images**: 400x400px, JPG format
- **Partner Logos**: 200x100px, PNG format with transparent background
- **OG Image**: 1200x630px, JPG format

## Temporary Solution

For development, you can use placeholder image services like:
- https://via.placeholder.com/800x800
- https://picsum.photos/800/800
- https://unsplash.com/ (for high-quality stock photos)

## Example Usage

```jsx
// In components, use Next.js Image component
import Image from 'next/image';

<Image
  src="/images/products/battery-pack-1.jpg"
  alt="EV Battery Pack"
  width={800}
  height={800}
  className="object-cover"
/>
```

Replace placeholder images with actual product photos, team headshots, and company logos for production.
